# Parallel Average Calculation using OpenMP

## Цель работы
Изучить основы параллельного программирования с использованием технологии OpenMP
на примере задачи подсчёта среднего значения элементов массива.

---

## Описание задачи
В работе реализовано вычисление среднего значения массива целых чисел
двумя способами:
- последовательным;
- параллельным с использованием OpenMP.

Для оценки эффективности параллельного подхода производится измерение
времени выполнения обоих вариантов.
